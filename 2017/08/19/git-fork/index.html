GitHub上fork的代码如何同步更新

2017-08-19
文章目录
  1. 1. 前言

github上有很多的代码，遇见心仪的项目总忍不住fork下来，但是过了一段时间以后原作者已经更新了提交，而我们的项目还是当时fork时的样子，该怎么办?

前言

一般习惯是fork项目以后，不去动原有的分支，如果要自己修改代码，则从当前master新建一个分支如my-dev，这样比较清爽。
